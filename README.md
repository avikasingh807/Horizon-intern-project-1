
# 🌐 Personal Portfolio Website

This is my personal portfolio website built using HTML and CSS.  
It showcases my skills, basic information, and contact details.

## 📌 Features
- Clean and simple user interface
- Navigation bar (Home, About, Skills, Contact)
- Profile image section
- Skills with progress bars
- Contact information

## 🛠️ Technologies Used
- HTML5
- CSS3

## 📁 Project Structure
portfolio/
│── index.html
│── style.css
│── images/
│     └── profile.jpg
## 🎯 Purpose
The purpose of this project is to create a personal portfolio website to showcase my web development skills and projects.

## 🚀 How to Run
1. Download or clone the project
2. Open the folder
3. Double-click on `index.html` to open in browser
